# Trellis for Codex

This plugin installs the Trellis planning and build harness into Codex as a project-local plugin.

Primary entrypoints in Codex are natural-language prompts such as `use Trellis`, `start Trellis`, `Trellis status`, and `Trellis build`.
This plugin includes Codex skill-discovery metadata so Trellis can be matched from those prompts. `AGENTS.md` reinforces the workflow, but the goal is native skill pickup rather than a fake slash-command surface.
