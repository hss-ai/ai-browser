# AI Browser v2.0 需求文档

> 版本: 1.0 | 日期: 2026-05-13 | 状态: 草案
> 目标: 在 v1.4.1 多模型对比平台基础上，全面提升功能性、美观度与用户体验

---

## 一、项目现状分析

### 1.1 已有能力

| 能力 | 说明 |
|------|------|
| 多模型广播 | 一键向 ChatGPT / Claude / Gemini / DeepSeek / 智谱清言 同时发送提问 |
| 多栏布局 | 支持 2~5 列并排展示，可拖拽调整列宽 |
| 多账号克隆 | 独立 partition 会话隔离，支持同一模型多开 |
| 指纹伪装 | 11 类浏览器指纹欺骗，7 套设备模板 |
| 代理配置 | HTTP / HTTPS / SOCKS4 / SOCKS5 代理 + 认证 |
| 会话持久化 | Cookies / 登录态跨重启保留 |
| 暗色主题 | 固定暗色 UI，紫/青/粉三色点缀 |

### 1.2 痛点与不足

| 编号 | 痛点 | 影响 |
|------|------|------|
| P1 | 无对话历史保存，刷新即丢失 | 用户无法回溯、无法复用优质回答 |
| P2 | UI 仅暗色，无法切换亮色 | 亮环境下视觉疲劳 |
| P3 | 面板顺序固定，不支持拖拽排序 | 用户无法按习惯排列模型 |
| P4 | 缺少回复对比工具，全靠肉眼 | 多模型对比效率低 |
| P5 | 无 Prompt 模板管理 | 高频优质 Prompt 每次需手动输入 |
| P6 | 仅支持 5 个固定 AI 模型 | 无法接入更多新兴模型 |
| P7 | 无自动更新机制 | 用户需手动下载新版本 |
| P8 | Webview 崩溃后无自动恢复 | 需要重启整个面板 |

## 二、新需求清单

### 模块 A: 对话管理（优先级: P0）

#### A1. 对话历史记录
- **描述**: 自动保存每次广播的提问和各模型的回复
- **功能要点**:
  - 按时间线存储，展示会话列表（日期 + 问题摘要）
  - 支持搜索历史对话（全文检索）
  - 支持按模型筛选对话
  - 支持收藏/置顶重要对话
  - 支持删除历史记录
- **存储方案**: 本地 SQLite 数据库（electron-store 扩展）
- **数据结构**:
  ```
  Session: { id, timestamp, question, tags, starred }
  Response: { id, sessionId, modelType, content, latency, rating }
  ```

#### A2. 对话导出
- **描述**: 将对话导出为多种格式
- **功能要点**:
  - 导出为 Markdown (.md)
  - 导出为 PDF（排版美观，含模型标签和评分）
  - 导出为 HTML（可分享的网页）
  - 支持单条导出或批量导出
  - 导出时保留代码高亮和格式

#### A3. Prompt 模板库
- **描述**: 内置 + 自定义 Prompt 模板管理
- **功能要点**:
  - 内置常用模板（翻译、代码审查、文案写作、学习助手等）
  - 支持创建/编辑/删除自定义模板
  - 支持模板分类（标签管理）
  - 模板支持变量占位符 `{{变量名}}`
  - 支持导入/导出模板（JSON 格式）
  - 快捷键快速唤起模板面板

---

### 模块 B: 智能对比（优先级: P0）

#### B1. 回复评分系统
- **描述**: 对各模型的回复进行多维度评分
- **功能要点**:
  - 自动采集响应时间（从发送到首字出现）
  - 支持用户手动评分（1~5 星）
  - 多维度评价标签：准确性 / 完整性 / 创意性 / 代码质量
  - 评分统计面板：各模型平均分、胜率排行
  - 支持「最佳回复」标记

#### B2. 回复差异对比
- **描述**: 将多个模型的回复进行可视化差异对比
- **功能要点**:
  - 并排 Diff 视图（类似代码 diff，高亮差异段落）
  - 关键词重合度分析（各模型回复的共性和差异点）
  - 回复长度/结构对比图表
  - 一键生成对比报告

#### B3. AI 回复摘要
- **描述**: 自动生成多模型回复的综合摘要
- **功能要点**:
  - 提取各模型回复的核心观点
  - 汇总共识点和分歧点
  - 生成「综合最优回答」（融合各模型优势）
  - 可选调用本地模型或外部 API 生成摘要

---

### 模块 C: UI/UX 升级（优先级: P0）

#### C1. 多主题系统
- **描述**: 支持多套视觉主题切换
- **功能要点**:
  - 内置 3 套主题: 深邃暗夜（当前） / 晨光亮色 / 赛博朋克
  - 支持自定义主题编辑器（主色、背景色、字体色等）
  - 主题跟随系统深浅色模式自动切换
  - 各面板标题色可独立自定义
  - 主题设置持久化保存

#### C2. 面板拖拽排序
- **描述**: 支持通过拖拽调整面板排列顺序
- **功能要点**:
  - 拖拽面板标题栏进行排序
  - 拖拽过程有流畅动画和占位提示
  - 排序结果持久化
  - 支持固定/解锁面板位置

#### C3. 灵活分屏布局
- **描述**: 支持更灵活的分屏方式
- **功能要点**:
  - 水平分屏（当前） + 垂直分屏 + 混合分屏
  - 预设布局模板: 2列 / 1上2下 / 十字四宫格 / 1主+侧栏
  - 拖拽分割线调整各区域大小
  - 布局模板可保存和切换

#### C4. 迷你模式 / 浮窗模式
- **描述**: 紧凑界面模式，适合多任务场景
- **功能要点**:
  - 迷你模式: 仅保留输入框和回复摘要，窗口缩小到 400x300
  - 浮窗模式: Always-on-top 小窗口，可拖拽到屏幕任意位置
  - 系统托盘图标，快速切换模式
  - 全局快捷键唤起/隐藏

#### C5. 界面动效优化
- **描述**: 增加流畅的界面过渡动画
- **功能要点**:
  - 面板展开/折叠动画
  - 主题切换过渡动画
  - Toast 通知进出场动画优化
  - 加载骨架屏 (Skeleton Loading)
  - 响应到达时的脉冲反馈

---

### 模块 D: 模型扩展（优先级: P1）

#### D1. 可配置模型管理
- **描述**: 支持用户自定义添加 AI 模型
- **功能要点**:
  - 提供模型配置界面: 名称、URL、图标颜色、CSS 选择器
  - 内置注入脚本模板（textarea / contenteditable / 自定义）
  - 支持导入社区共享的模型配置包
  - 模型启用/禁用开关
  - 预设热门模型快速添加:
    - Grok (xAI)
    - Perplexity
    - Mistral Chat
    - 通义千问
    - Kimi (月之暗面)
    - 文心一言

#### D2. 本地模型接入
- **描述**: 支持接入本地部署的 AI 模型
- **功能要点**:
  - Ollama 本地模型集成
  - LM Studio 本地模型集成
  - 自定义 OpenAI 兼容 API 端点
  - 本地模型状态监控（运行中/离线）

---

### 模块 E: 效率工具（优先级: P1）

#### E1. 快捷键系统
- **描述**: 可自定义的全局和应用内快捷键
- **功能要点**:
  - 默认快捷键方案:
    - `Ctrl+Enter` 发送广播
    - `Ctrl+1~5` 切换列数
    - `Ctrl+L` 清空输入
    - `Ctrl+Shift+T` 打开模板
    - `Ctrl+,` 打开设置
    - `Ctrl+Shift+F` 搜索历史
  - 支持自定义绑定
  - 快捷键冲突检测
  - 快捷键速查面板

#### E2. 剪贴板增强
- **描述**: 增强剪贴板交互体验
- **功能要点**:
  - 全局快捷键一键粘贴并发送
  - 选中文本右键直接发送到指定模型
  - 剪贴板历史（最近 20 条）
  - 自动检测剪贴板中的代码/URL 并提示发送

#### E3. Markdown 渲染增强
- **描述**: 对 Webview 中的回复进行 Markdown 优化渲染
- **功能要点**:
  - 注入代码语法高亮样式
  - 表格格式化渲染
  - 数学公式渲染 (KaTeX)
  - 流程图渲染 (Mermaid)
  - 链接预览卡片

---

### 模块 F: 工程质量（优先级: P1）

#### F1. 自动更新
- **描述**: 应用启动时自动检查并提示更新
- **功能要点**:
  - 基于 GitHub Releases 检查新版本
  - 支持后台下载 + 安装提示
  - 更新日志展示 (Changelog)
  - 可选自动/手动更新模式

#### F2. Webview 健壮性
- **描述**: 提升 Webview 稳定性和容错能力
- **功能要点**:
  - Webview 崩溃自动检测 + 自动重载
  - 网络断开重连机制
  - 页面加载超时处理（30s）
  - 内存占用监控 + 超阈值自动释放
  - 加载失败友好错误页面

#### F3. 数据安全
- **描述**: 加强本地数据安全性
- **功能要点**:
  - 敏感配置加密存储（AES-256）
  - 会话数据可选密码保护
  - 清除数据功能（一键清除所有缓存和历史）
  - 启动时可选锁屏密码

---

### 模块 G: 高级特性（优先级: P2）

#### G1. Prompt 工作流
- **描述**: 支持 Prompt 链式编排
- **功能要点**:
  - 可视化 Prompt 链编辑器
  - 支持条件分支（根据上一轮回复选择下一步）
  - 支持变量传递（上一轮输出作为下一轮输入）
  - 预设工作流模板（翻译优化链、代码审查链等）
  - 工作流执行日志

#### G2. 多人协作
- **描述**: 支持小团队共享对话和 Prompt
- **功能要点**:
  - 对话分享链接（端到端加密）
  - Prompt 模板市场（社区共享）
  - 团队工作区
  - 实时协作光标（查看队友正在对比的内容）

#### G3. AI 回复质量报告
- **描述**: 定期生成 AI 模型使用报告
- **功能要点**:
  - 各模型使用频率统计
  - 平均响应时间趋势
  - 用户评分趋势
  - 模型表现排行
  - 导出月度/周度报告

## 四、非功能性需求

| 类别 | 要求 |
|------|------|
| 性能 | 启动时间 < 3s；5 面板同时加载内存 < 800MB |
| 兼容性 | Windows 10+ / macOS 12+ / Ubuntu 22.04+ |
| 安全性 | 本地数据加密；无遥测数据上传；HTTPS 强制 |
| 可访问性 | 支持键盘全操作；屏幕阅读器兼容；高对比度模式 |
| 国际化 | 中文 / 英文双语，可扩展更多语言 |
| 稳定性 | Webview 崩溃率 < 0.5%；连续运行 8h 无内存泄漏 |
